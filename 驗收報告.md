## 六、建議部署方案（快速上線）

### 方案 A：Render（推薦，免費）
1. 將專案推上 GitHub
2. Render.com → New Web Service → 連接 GitHub
3. Build Command：`npm install`
4. Start Command：`node server.js`
5. 自動提供 HTTPS，免費方案足夠 MVP 使用

### 方案 B：Railway（備選）
- 類似 Render，UI 更簡潔
- 免費額度每月 $5 美金

### 資料庫升級（部署後第一週）
- 安裝 `better-sqlite3`，將 `submissions.json` 替換為 SQLite 本地檔案
- 雲端平台掛載 persistent disk，資料不會因重啟消失
